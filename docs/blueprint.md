# **App Name**: Kim's Portfolio

## Core Features:

- Navbar: Sticky, translucent navigation bar with links to different sections of the portfolio.
- Hero Section: Hero section with headline, subheadline, download resume button, and view work button to introduce Josephus Kim L. Sarsonas.
- About Section: About section showcasing Josephus Kim L. Sarsonas's details, bio, and personal qualities.
- Skills Section: Skills section displaying various skills grouped by category using shadcn cards.
- Portfolio Section: Portfolio section showcasing projects with hover animation and modal detail view, demonstrating Josephus Kim L. Sarsonas's work.
- Experience Section: Experience section utilizing a timeline component to present Josephus Kim L. Sarsonas's professional experience.
- Contact Section: Contact form using shadcn inputs along with buttons for LinkedIn, GitHub, and email to facilitate contact.

## Style Guidelines:

- Primary color: Fluent blue (#0078d7) to align with the requested design, symbolizing trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean and modern feel.
- Accent color: Slightly darker shade of blue (#005a9e) to provide contrast for CTAs and highlights without being distracting.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used to provide a modern, neutral, and clean appearance appropriate for both headlines and body text.
- lucide-react icons will be used throughout the site to maintain a consistent and professional visual language.
- The layout will follow a clean, minimal design with rounded corners (`rounded-2xl`) and soft shadows (`shadow-md`) to provide depth and visual appeal.
- Framer Motion will be used for subtle animations such as section fade-in on scroll and card hover lift, enhancing user experience.
